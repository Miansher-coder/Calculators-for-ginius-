How This Calculator Works:

🔹 Numerator → The number above the line (top part of fraction).
🔹 Denominator → The number below the line (bottom part of fraction).

Example:
If you enter 2 / 3
The 2 is the numerator and 3 is the denominator.

Operations:
➕ Addition → Cross multiply and add.
➖ Subtraction → Cross multiply and subtract.
✖ Multiplication → Multiply top × top and bottom × bottom.
➗ Division → Multiply by reciprocal (flip second fraction).

Mixed Fraction Option:
If result is improper (like 7/3), enabling this will display it as:
2 1/3
